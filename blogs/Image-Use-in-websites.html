Image Use in websites

21 August 2021

When I started creating basic Html and CSS websites, I didn't pay much attention to small things which had a crucial role in websites; Images.

During practice, I used to pick any picture I want from the internet and put it on my website as it was for my personal use and not for any commercial use. But when I started doing internships and freelancing, I got to know about copyright issues. We can't take any image you want from the internet for commercial purposes; we must check for copyright.

With experience, I can tell two ways you can get no copyright images.

1. Google Images

When searching for images in google, go in tools and check for usage rights, you will see two options,

Creative Commons licenses

It is a license issued by the copyright owner to allow anyone in the world to use their copyright work in any manner consistent with that license.

Commercially licensed images

Images used with the intent to sell a product, raise money, promote or endorse something.



If you want to use the image for commercial purposes you should go for a common creative license as it is free to use and without copyright.

2. Copyright Free Image Websites

Another way to get copyright-free images is to go to listed popular websites:

1. Unsplash

2. Pixabay

3. Pexels
